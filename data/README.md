# Data

This directory contains the compound data. It consists of an overview file and four subdirectories.
The subdirectories contain files with the compounds in their sentence context.

## Overview

An overview of all compounds annotated for branching direction and internal semantic relationships is found in the file *compound_overview.csv*.

## EN_general

Data files for English general language.

## EN_scientific

Data files for English scientific language.

## GER_general

Data files for German general language.

## GER_scientific

Data files for German scientific language.
