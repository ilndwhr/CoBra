# CoBra (Compound Branching Resource)

This repository contains data, tools and (pre)processing scripts for the project CoBra (Compound Branching Resource).
The resource was presented at the Ninth Workshop on Universal Dependencies (UDW 2026), held in conjunction with LREC 2026.
